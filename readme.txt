Dit is alle python-code, trainingsdata en testdata die voor dit project gebruikt zijn.

in de main.py file wordt eigenlijk al het belangrijke gedaan, met hulp van de train_lego.py file.

Alle bestanden die met 'test' beginnen zijn niet gebruikt in het uiteindelijke programma, maar laten wel zien hoe het programma gegroeit is.
